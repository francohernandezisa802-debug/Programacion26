Proyecto Final de Programación

Este proyecto reúne dos aplicaciones desarrolladas utilizando HTML, CSS y JavaScript.

El objetivo es aplicar conceptos básicos de programación como variables, operaciones matemáticas, estructuras condicionales y captura de datos.

💪 Calculadora de Índice de Masa Corporal (IMC)

Permite calcular el Índice de Masa Corporal a partir del peso y la altura del usuario.

Conceptos utilizados:

  • Variables
  • Condicionales (if)
  • Operaciones matemáticas
  • Manipulación del DOM

💧 Calculadora de Hidratación Diaria

Calcula la cantidad recomendada de agua diaria según el peso corporal del usuario.

Conceptos utilizados:

  • Variables
  • Funciones
  • Operaciones matemáticas
  • Validación de datos
